Python Data Type

In Python we have different types of data such as: integer, strings, etc.


int: Integer is a whole number that does not contain any decimals. For ex: 8, 100, etc.

float: Float is the opposite of an integer, because it is a number that containes a decimal. For ex: 9.6, 300.10, etc.



string: String is a letter, word, sentence or a collection of sentences. For ex: 'Hello World'

boolean: Boolean consists of two values True and False. It is used to check if the statement is True or False

Python Operators


Python Arithmetic Operators


Arithmetic Operators are operators that take numerical values as their operands and return a single numerical value.

Operator Description Example
+ Adding operands a + b = 10
- Subtracts operands a - b = 5
* Multiplies operands a * b = 20
/ Quotient of division of left operand by right operand a / b = 1.7 (float)
// Quotient of division of left operand by right a // b = 3 (int)
% Remainder of division of left operand by right operand a % b = 1
** Left operand raised to the power of right operand a ** b = 9



Python Relational Operators


Relational Operators check the relationship between two operands if they are True or False

Operator Description Example
== Equal to (a == b) is False
!= Not equal to (a != b) is True
> Greater Than (a > b) is True
< Less Than (a < b) is False
>= Greater than or equal to (a >= b) is True
<= Less than or equal to (a >= b) is False
